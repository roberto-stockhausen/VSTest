Tags de HTML

"O que as coisas entre "<" e ">" significam?"

html

Indica o início e fim do texto HTML a ser interpretado

head

indica o espaço onde será escrito codigo para o cabeçalho do documento

title

define o título que aparecerá na aba da página

body

indica o espaço para escrever o código HTML que será visível na página

footer

indica o espaço de rodapé da página

H

h adiciona um título na página. Pode ter diferentes tamanhos representados por "h1"(Maior) até "h6"(Menor)

p

"p" é usado para inserir parágrafos de texto

b

b é usado para texto em negrito

i

i é usado para texto em italico

br

br é usado para
Pular um parágrafo

hr

hr é usado para

Criar essa linha horizontal

span

normalmente usado para textos curtos

pre

pre é usado para textos pré-formatados

img

img é usado para adicionar imagems tanto da web quanto de arquivos do diretório principal.

audio

audio é usado para adicionar som no website

a

a é usado para inserir Links

video

video é usado para adicionar videos ao website

form

form é usado para definir o espaço de um formulário, usado para obter informações

input

usado para receber input do usuário

textarea

usado para definir o espaço onde o usuário pode digitar o input
